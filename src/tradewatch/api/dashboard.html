About

TradeWatch is a big-data trade-surveillance stack — an Apache Kafka → FastAPI speed layer for sub-10ms real-time anomaly alerts, with an Apache Spark / PySpark and Hadoop (HDFS + MapReduce) scale layer for backtesting and distributed detection. Seven detectors combine statistical, behavioural and machine-learning methods, each alert carrying an explainable reason and severity.
